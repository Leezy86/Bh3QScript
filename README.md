# 崩坏3按键精灵脚本

*不支持手机*

目前只有战场脚本

基于崩坏3pc端制作，支持1080P和720P分辨率，其他分辨率可能会让脚本无法启动

所有键位设置为默认设置，如果已经修改过，需要恢复成默认按键，你也可以自己改脚本设置成你的键位

**游戏最好是设置成120帧**

## 比较好用的特性

大部分脚本支持：

* **自动判断凹点（如猫中是否生效，是否破盾）**

* **自动判断出分，识别战场时间**

* **没有出分则自动重开，真正的睡觉即可出分**

* **未来考虑支持模拟器，如果官方PC端禁止按键精灵的话**

* **长期更新（完全看兴趣，极有可能鸽鸽鸽）**
